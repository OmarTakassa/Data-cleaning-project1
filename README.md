# Data-cleaning-project1
## Overview

In this project, I demonstrate **how to clean data** from a table called `layoffs` using **PostgreSQL**.  

Real-world data is often messy, inconsistent, or incomplete. Cleaning it is a **crucial step** before any analysis, reporting, or machine learning tasks.  

This project focuses on:  

- **Removing duplicates**  
- **Standardizing text fields**  
- **Converting data types**  
- **Handling null or inconsistent values**  

With proper cleaning, the `layoffs` table becomes **consistent, reliable, and ready for analysis**.
